**This is the missing organ in today's AI — and in the current arifOS federation.**

### 4a. What It Is

The Reason Engine sits between the state engines (Layer 2) and governance (Layer 4). It does not speak. It does not guess. It does not pretend. It reasons **over** the state engines.

### 4b. Sub-Organs

The Reason Engine has three sub-organs, forming the **Planner → Reasoner → Critic** chain:

```
State Engines
    ↓
Planner — decides what evidence is needed
    ↓
EGS/WEALTH/WELL — state engines execute queries
    ↓
Reasoner — builds competing hypotheses
    ↓
Critic — attempts falsification
    ↓
Governance
```

|--------|--------------|
| **Input** | User question or task |
| **Output** | Structured query plan against state engines |

| Aspect | Specification |
|--------|--------------|
| **Input** | Structured state from state engines |
| **Output** | Set of competing hypotheses with evidence + confidence |
| **Method** | Generates multiple internally consistent interpretations, each with support + weight |
| **Example** | "Charge scenario A (early migration, high volume) vs Scenario B (late charge, limited)" |

| Aspect | Specification |
