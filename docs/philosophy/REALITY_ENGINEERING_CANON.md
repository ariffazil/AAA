## §4 Layer 3: Reason Engine

**This is the missing organ in today's AI — and in the current arifOS federation.**

### 4a. What It Is

The Reason Engine sits between the state engines (Layer 2) and governance (Layer 4). It does not speak. It does not guess. It does not pretend. It reasons **over** the state engines.

### 4b. Sub-Organs

The Reason Engine has three sub-organs, forming the **Planner → Reasoner → Critic** chain:

```
State Engines
    ↓
Planner — decides what evidence is needed
    ↓
EGS/WEALTH/WELL — state engines execute queries
    ↓
Reasoner — builds competing hypotheses
    ↓
Critic — attempts falsification
    ↓
Governance
```

#### Planner (Evidence Design Agent)

| Aspect | Specification |
|--------|--------------|
| **Input** | User question or task |
| **Output** | Structured query plan against state engines |
| **Method** | Decomposes question into sub-queries, ordered by dependency |
| **Example** | "Underfill risk?" → `[get_trap_model, get_charge_model, get_pressure_history, get_seal_capacity]` |
| **Status** | ❌ NOT YET IMPLEMENTED — partial precursor in `arif_route(mode=auto)` |
| **Home** | `AAA/agents/planner/` (333-AGI domain) |

#### Reasoner (Hypothesis Builder)

| Aspect | Specification |
|--------|--------------|
| **Input** | Structured state from state engines |
| **Output** | Set of competing hypotheses with evidence + confidence |
| **Method** | Generates multiple internally consistent interpretations, each with support + weight |
| **Example** | "Charge scenario A (early migration, high volume) vs Scenario B (late charge, limited)" |
| **Status** | ❌ NOT YET IMPLEMENTED — partial precursor in `geox_evidence(mode=abduct)` |
| **Home** | Per-organ — `GEOX/src/geox_core/engines/reasoner/`, `WEALTH/src/engines/reasoner/`, etc. |

#### Critic (Falsification Engine)

| Aspect | Specification |
|--------|--------------|
| **Input** | Competing hypotheses from Reasoner |
| **Output** | Ranked hypotheses with falsification attempts + scores |
| **Method** | Tests each hypothesis against observed data, physical consistency, mass balance, pressure compatibility |
| **Example** | "Scenario A violates observed pressure gradient — falsification score 0.7. Scenario B passes all checks — score 0.2" |
| **Status** | ❌ NOT YET IMPLEMENTED — partial precursor in `arif_think(mode=critique)` and `geox_evidence(mode=contradict)` |
| **Home** | Domain-Critic per organ + Meta-Critic in `arifOS/arifosmcp/critique/` |

### 4c. Reason Engine Invariant

> **Reasoning owns hypotheses. It does not own authority. It does not own language. It does not own state. It proposes; governance disposes.**

## §6 Layer 5: LLM — Language Organ

### 6a. What It Is

The LLM is the **language organ** — the mouth, not the mind. It consumes governed state from lower layers and produces human-readable language. It does not invent Earth, capital, or human state.

### 6b. Hard Constraints

1. **No free-form reality claims.** Every statement about depth, volume, pressure, capital, or human state must trace to a state engine query.
2. **No epistemic label upgrade.** OBSERVED stays OBSERVED. HYPOTHESIS stays HYPOTHESIS.
3. **No authority claims.** The LLM cannot say "I approve this" — that belongs to Layer 4.
4. **Explicit uncertainty.** Every claim must surface confidence, intervals, or scenario weights.
5. **Decision gating.** Irreversible actions → route to arifOS 888_HOLD → human approval.
6. **Auditability.** Every answer that uses state engines produces a receipt.

### 6c. LLM Invariant

> **Language owns communication. It does not own reality, authority, state, or hypotheses. It is the mouth, not the mind.**

## §8 How This Prevents AI Worship

The danger ChatGPT identified:

> *"If AI becomes accurate enough, humans may confuse 'high-fidelity representation' with reality itself."*

The 6-layer stack prevents this structurally:

| Worship Vector | Prevention |
|---------------|------------|
| "AI is reality" | Layer 1 is not a model — it's reality. State engines (Layer 2) are interfaces, not reality itself. |
| "AI has authority" | Layer 4 owns authority exclusively. LLM cannot issue judgments. |
| "AI is always right" | Layer 3 proposes competing hypotheses + falsification scores. No single answer is guaranteed. |
| "AI sees the world" | Vision models (Layer 2 extension) add perception but not epistemic discipline or governance. |
| "AI is the truth" | Every Layer 2 output carries epistemic label + uncertainty + provenance. Truth is labelled, not asserted. |

**The structural answer:**

> Humans worship confident language. They do not worship structured state engines with explicit uncertainty bands and provenance chains. An LLM that says "P50: 2450m, P10: 2350m, P90: 2550m, source: W_01 deviation survey" is not worshippable. An LLM that says "The depth is 2450 meters" is.

The 6-layer stack forces the LLM into the first form and prevents the second.
