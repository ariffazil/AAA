# AAA — Repo Role Boundary

> Role: Federation Observer / Orchestration Intelligence
> Anti-Sink Responsibility: Monitor cross-agent drift, dashboard topology risk

---

## Invariant Role

AAA is the **control plane and A2A mesh** of the federation. It:
- Exposes agent-to-agent communication (A2A gateway, port 3001)
- Hosts the operator dashboard (Cockpit)
- Provides observability infrastructure (Prometheus, Grafana)
- Manages agent identity and seed configuration
- Routes federation-wide governance signals

## Must Never Become

- A governance kernel (that is arifOS)
- An economic engine (that is WEALTH)
- An Earth intelligence (that is GEOX)
- A forge/mutation engine (that is A-FORGE)
- A human vitality mirror (that is WELL)

## Anti-Sink Diagnostics Dashboard

When topology diagnostics are wired, AAA should display:
- Per-agent agency compression score
- Cross-agent role diversity delta
- Federation-wide extractive drift trend
- Beautiful ones risk flag (agents producing output without action)
- Sovereignty integrity gauge
