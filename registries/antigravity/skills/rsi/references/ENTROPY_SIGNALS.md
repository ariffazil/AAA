# Entropy Signals

When performing an RSI pass, measure and document the following signals of system entropy:

1. **Duplication**: Repetitive logic or copy-pasted blocks.
2. **Boundary Bleed**: Separation of concerns violations (e.g., UI logic in data layer).
3. **Dead Code**: Unused functions, stale comments, or deprecated APIs.
4. **Cognitive Load**: Functions exceeding 50 lines or passing more than 4 parameters.
5. **Test Coverage Gaps**: Critical paths lacking validation.
