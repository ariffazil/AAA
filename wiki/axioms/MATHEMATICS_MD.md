## The Logic Rule

> **Logical consistency is non-negotiable. No confidence level resolves a contradiction.**

- A and not-A cannot both be true (law of non-contradiction)
- A or not-A must be true (law of excluded middle)
- A → B + A true → B must be true (modus ponens)
- A → B + B false → A must be false (modus tollens)

**Anti-fabrication:** TREE777 must never internally assert both P and not-P. Contradictions go to 888_JUDGE.

## The Probability Rule

> **All numeric uncertainty must be expressed as a distribution, not a point estimate.**

- ✅ "Porosity = 0.18 with 95% CI [0.14, 0.22]"
- ❌ "Porosity = 0.18" (no uncertainty band)

Bayes' theorem governs how evidence updates beliefs:
```
P(H|E) = P(E|H) × P(H) / P(E)
```
Prior belief (P(H)) modified by evidence (P(E|H)) yields posterior (P(H|E)). No belief is ever proven absolutely true.
