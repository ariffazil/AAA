# AXIOM: Language — Semantics, Syntax, Semiotics, Communication

> **Dimension:** 0 — Foundational (Root)
> **Grounds:** All TREE777 pages, all prompts, all skill descriptions, all inter-agent communication.
> **Invariant:** Language is the medium; it is not the truth itself.

---

## What This Axiom Covers

Language is the medium through which all intelligence encodes, transmits, and interprets meaning. Core domains:
- **Semantics** — meaning of signs and symbols; reference and sense
- **Syntax** — formal rules of combination; grammatical structure; well-formedness
- **Pragmatics** — context-dependence; implicature; speech acts; intention
- **Semiotics** — signs (index, icon, symbol); Peirce's triadic model
- **Communication theory** — Shannon's model (sender, channel, receiver, noise, entropy)
- **Formal languages** — regex, context-free grammars, programming languages
- **Ontology** — naming conventions; controlled vocabularies; namespace management

## The Precision Rule

> **Ambiguity is the enemy of intelligence. Define terms before using them.**

**Anti-fabrication:** "Health" in WELL ≠ "Health" in VPS ≠ "Health" in arifOS. Every skill page must define ambiguous terms in a `Definitions` section.

## The Semiotics Triangle

```
    THOUGHT ─────── REFERENCE
        │╲             ╱
        │  ╲  inter-  ╱
        │    ╲prets   ╱
        ▼      ╲    ▼
      SYMBOL ──────▶ OBJECT
```

- **Symbol:** The word or notation ("entropy", "ΔS")
- **Reference:** The actual thing denoted (thermodynamic quantity)
- **Thought:** The mental interpretation

**Failure mode:** When symbol and reference diverge — when an agent uses "entropy" but means something different from the thermodynamic definition. This is the source of most inter-agent miscommunication.

## The Shannon Channel Rule

```
C = B × log₂(1 + S/N)
```
Maximum bits/second over a channel with bandwidth B and signal-to-noise ratio S/N.

**Practical implications for TREE777:**
- Context window is finite; progressive disclosure is not optional
- F1 AMANAH and F4 CLARITY reduce noise in the channel
- The 5K/skill budget is a channel capacity management mechanism

## The Formal Language Rule

> **Code is language with execution semantics. It must be grammatically valid before it can be evaluated.**

- Python must pass `ruff` and `mypy` before considered syntactically valid
- TypeScript must compile before considered well-formed
- JSON must parse before used as data
- SQL must be syntactically valid before execution

**Anti-fabrication:** A skill that generates untested code is not trustworthy.

---

## The Translation Invariance Rule

> **Meaning must be preserved across transformations. If the meaning changes, the transformation failed.**

- Python → TypeScript: algorithm must be preserved
- English → Malay: definition must be semantically equivalent
- JSON encode → decode: intent must be preserved
- raw/notes → wiki/skills: evidence must be preserved

**Anti-fabrication:** Test translation by asking: would the same inputs produce the same outputs?

## Summary

**Language is the vehicle of all representation.** It is not truth itself, but it is the vehicle by which truth is transmitted.

TREE777 must always respect:
- Semantics over noise — define terms before using them
- Syntax over ambiguity — grammatically valid before semantically meaningful
- Reference over symbol-worship — the symbol is not the thing
- Shannon limits — channel capacity is finite; progressive disclosure is mandatory
