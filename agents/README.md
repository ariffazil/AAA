# agents/ — AGENT CONFIGURATION DIRECTORY

> This directory contains agent operational configs, not A2A protocol cards.
> 
> Structure:
> - `_lanes/` — HEXAGON lane agent cards (333-AGI, 555-ASI, 888-APEX, 777-FORGE)
> - `_external/` — external harness configs (kimi-code, opencode, etc.)
> - `_docs/` — agent documentation and zen init
> - `_archive/` — retired agent configs
> - `agent_cards/` — exported agent card snapshots
> 
> **Canonical A2A cards:** `a2a-server/agent-cards/`
> **Canonical agent registry:** `registries/agents.yaml`
