# ABKSS Naming Framework

## What Happened

ARIF created a systematic nomenclature for Sabah Basin geological features. ABKSS (the framework) unified how wells, prospects, and geological bodies are named across the entire basin.

## Before

Every geologist named features ad-hoc. Same structure had different names. Communication was lossy.

## After

A unified naming discipline. AMANAH (trust) entered: because naming is standardized, reports can be trusted across teams. AKAL and PRESENT were already present (geologists observed and named). AMANAH was the addition — the commitment to a shared vocabulary.
