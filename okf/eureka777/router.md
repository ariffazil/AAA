# EUREKA777 Activation Router

## Trigger Conditions

A EUREKA777 entry is created when ONE of these conditions fires:

1. **New phenomenon observed** → class: DISCOVERY
2. **Hypothesis test completes** (falsified or survived) → class: FALSIFICATION
3. **Two previously separate domains connect** → class: UNIFICATION
4. **Narrative pole flips** (X was Y → actually ¬Y) → class: INVERSION
5. **AMANAH floor triggers HOLD or VOID** → class: REFUSAL
