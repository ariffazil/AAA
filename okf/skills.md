# 33-Skill Copilot Operating Model

## The Architecture

If normal Copilot is "help me with this task," the 33-skill Copilot becomes:
"Help me understand reality, remember reality, make decisions about reality, coordinate work across reality, and protect the human making those decisions."

## Layer Breakdown

### Memory Layer — What is true about history?
Skills: 4 Truth & Memory skills
- Evidence classification (OBS/DER/INT/SPEC)
- Session continuity
- Knowledge preservation
- Decision traceability

### Earth Layer — What is true about the Earth?
Skills: 5 GEOX skills
- Subsurface evidence mapping
- Seismic interpretation
- Basin analysis
- Petrophysical evaluation
- Prospect risk assessment

### Capital Layer — What is true about value?
Skills: 5 WEALTH skills
- NPV/IRR/EMV computation
- Monte Carlo simulation
- Market intelligence
- Institutional diagnostics
- Capital wisdom synthesis

### Human Layer — What is true about the human?
Skills: 7 WELL skills
- Substrate classification
- Vitality assessment
- Fatigue detection
- Dignity guarding
- Repair/recovery check
- Cognitive load awareness
- Homeostasis assessment

### Operations Layer — What is true about the work state?
Skills: 9 AAA skills
- Work coordination
- Task state tracking
- Agent handoff
- Approval routing
- Blocker detection
- Risk reporting
- Session handoff
- Skill unification
- Cross-agent orchestration

### Cognitive Geometry — How to think across all layers
Skills: 3 ATLAS333 skills
- Paradox activation
- Tension vector navigation
- Contour reasoning

## Total: 33 Skills

## End-to-End Workflow (PETRONAS Example)

1. **check-before-start** — Has anyone already studied this play?
2. **subsurface-evidence-map** — Gather seismic, wells, biostrat, grav-mag, papers
3. **uncertainty-and-alternatives** — What interpretations remain possible?
4. **resource-allocation-review** — Is further work justified?
5. **decision-record** — Why are we proceeding?
6. **project-memory** — Preserve context
7. **operator-dashboard-summary** — What is next?
