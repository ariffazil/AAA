## Strategic Position

OpenAI Agents SDK is the best simple agent runtime to wrap. arifOS should use it as a fast execution substrate, not as sovereign authority.

```
OpenAI Agents SDK = clean agent loop
arifOS = law before/around/after the loop
```

## Integration Pattern

```
OpenAI guardrail failure → arifOS HOLD/VOID
```

## Key Distinction

OpenAI guardrails validate inputs/outputs. arifOS governs consequences.

| Guardrail Type | What It Checks | Example |
|----------------|---------------|---------|
| OpenAI (runtime) | Input/output format, topic restriction | Block profanity, block PII |
| arifOS (constitutional) | Reversibility, truth, sovereignty, blast radius | Block irreversible deploy without F13 |

## Boundary Rules

- OpenAI SDK may execute the agent loop. arifOS sets the loop's legal bounds.
- OpenAI tool calls get an arifOS lease before invocation.
- OpenAI handoffs get an arifOS scope transfer.
- OpenAI traces become VAULT999 receipts.
